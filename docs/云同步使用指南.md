# SnippetsHub 云同步功能使用指南

## 概述

SnippetsHub 云同步功能允许您将代码片段上传到 GitHub Gist 或 Gitee，方便在不同设备间分享和备份代码。

## 支持的平台

### 1. GitHub Gist
- 全球最流行的代码片段分享平台
- 支持公开和私密 Gist
- 自动语法高亮
- 支持评论和 fork

### 2. Gitee 代码片段
- 国内访问速度快
- 支持公开和私有
- 适合国内用户使用

## 配置步骤

### GitHub 配置

1. **获取 Personal Access Token**
   - 访问 [GitHub Token 创建页面](https://github.com/settings/tokens/new?scopes=gist)
   - 选择 `gist` 权限
   - 点击 "Generate token"
   - 复制生成的 Token（格式：`ghp_xxxxxxxxxxxx`）

2. **在 SnippetsHub 中配置**
   - 进入 **设置** → **云同步**
   - 选择 **GitHub** 标签
   - 粘贴 Token 到输入框
   - 选择默认可见性（公开/私密）
   - 点击 **保存配置**

3. **验证连接**
   - 保存后会自动验证 Token
   - 连接成功会显示您的 GitHub 用户名和头像

### Gitee 配置

1. **获取私人令牌**
   - 访问 [Gitee Token 创建页面](https://gitee.com/profile/personal_access_tokens/new)
   - 选择 `gists` 权限
   - 点击 "提交"
   - 复制生成的令牌

2. **在 SnippetsHub 中配置**
   - 进入 **设置** → **云同步**
   - 选择 **Gitee** 标签
   - 粘贴令牌到输入框
   - 选择默认可见性（公开/私密）
   - 点击 **保存配置**

## 上传代码片段

1. 确保已配置并连接到平台
2. 在 **上传代码片段** 区域选择要上传的代码片段
3. （可选）修改 Gist 描述和文件名
4. 点击 **上传** 按钮
5. 上传成功后会显示 Gist 链接

## 功能特点

### 自动文件扩展名
系统会根据代码语言自动添加正确的文件扩展名：
- JavaScript → `.js`
- Python → `.py`
- Java → `.java`
- C++ → `.cpp`
- 等等...

### 上传历史
- GitHub 用户可以查看最近 5 个上传的 Gist
- 点击可直接跳转到 Gist 页面

### 可见性控制
- **公开**：任何人都可以查看
- **私密/私有**：只有您可以查看（通过直接链接也可访问）

## 注意事项

⚠️ **安全提示**
- Token 存储在本地浏览器中（localStorage）
- 不要在公共计算机上保存 Token
- 定期更新和轮换 Token

⚠️ **限制说明**
- GitHub API 有速率限制（每小时 5000 次请求）
- Gitee API 也有相应的限制
- 大型代码文件可能需要较长上传时间

## 常见问题

### Q: Token 无效怎么办？
A: 请确保：
1. Token 有正确的权限（gist 或 gists）
2. Token 未过期
3. 网络连接正常

### Q: 能否批量上传？
A: 当前版本仅支持单个片段上传，未来版本将支持批量操作。

### Q: 上传的 Gist 能否删除？
A: 需要在 GitHub/Gitee 网站上手动删除。

### Q: 可见性设置在哪里？
A: 在配置 Token 时设置默认可见性，每次上传时使用该设置。

## 技术支持

如果遇到问题，请：
1. 检查浏览器控制台错误信息
2. 确认网络连接正常
3. 验证 Token 权限和有效性
4. 联系开发者获取帮助
