## [1.0.0] - 2026-02-05

### ✨ 新增功能

#### 代码片段管理
- 💻 智能代码片段管理系统
- 🔍 高级搜索和过滤功能
- 📁 文件夹和标签组织
- ⭐ 收藏功能
- 📤 批量导出（ZIP、JSON、Markdown）

- 📝 实时预览编辑器
- 🛠️ 丰富的编辑工具栏
- 📑 目录大纲导航
- 💬 浮动格式化工具栏
- 🔄 滚动同步
- 📤 导出功能
- 📜 历史记录

#### TODO任务管理
- ✅ 现代化任务管理界面
- 🏷️ 标签和优先级
- 📅 截止日期和提醒

- 📈 统计数据

- 🎯 Monaco Editor集成

- ⌨️ 全局快捷键支持

### 🎯 界面优化
- 现代化的用户界面设计
- 流畅的动画效果

### ⚡ 性能优化
- 虚拟滚动列表
- 懒加载组件

- 代码分割
- 资源优化

### 🔧 技术栈
- **前端**: Vue 3 + Vite + TypeScript
- **桌面**: Tauri 2.0
- **状态管理**: Pinia
- **编辑器**: Monaco Editor
- **图标**: Lucide Vue Next
- **样式**: 原生CSS + CSS变量

### 📦 平台支持
- macOS (Intel + Apple Silicon)
- Windows (x64)
